# aliyun-traffic-control

一键安装：

```bash
curl -sSL https://raw.githubusercontent.com/kingsnakerrr/aliyun-traffic-control/refs/heads/main/install.sh -o install.sh && bash install.sh
```

安装后输入：

```bash
cdt
```

功能：启动/停止/重启 TG Bot，开启/关闭保活，立即执行保活检查，查看最近20行日志，查看实时日志，完全卸载程序。

注意：不要把真实 AccessKey 和 Bot Token 提交到公开仓库。仓库里只放 `traffic_control.py.template`，安装时再写入真实配置。
